## 1. サービス概要

「キャリサポ」は、退職・学習・転職・スキルアップを一気通貫で支援するキャリア支援サービス。
1人の顧客から複数サービスで収益化できる構造が特徴。

### 運営体制
- 運営者: 自分ひとり(売上/業務次第でバイト採用予定)
- LP・公式LINE: 作成済み
- 現在フェーズ: スタートしたて
- SNS: Threadsのみ(@ogawa_iju)
- **公式LINE**: https://lin.ee/tpWHOuh
- **LP**: https://ogaignbr.github.io/carisapo/
- **Threads ID**: ogawa_iju
- 今後の検討: Meta/LINE広告、Instagram、TikTok(動画工数が課題)

## 2. サービスラインナップ

### ① 退職サポート
- **内容**: 失業保険などの申請サポート(外注トスアップ)
- **初期費用**: 10〜20万円
- **報酬**: 5〜10万円/件
- **ターゲット**: 会社員、仕事に不満、最近辞めた人

### ② 学習サポート
- **内容**: 就労移行支援(外注トスアップ)
- **初期費用**: 0円
- **報酬**: 2万円/月(通所している限り継続)
- **ターゲット**: 無職、主婦(障害の有無は問わない)
- **顧客メリット**: 最大24ヶ月間、毎月3万円を受給。費用なし
- **重要制約**: **定員最大15名**・仙台市内から片道2時間圏内限定(告知時は非公開)

### ③ 転職サポート
- **内容**: 企業への紹介(人材紹介)
- **初期費用**: 0円
- **報酬**: 20〜70万円/件
- **ターゲット**: 無職、転職希望、アルバイト、お祝い金目的
- **顧客メリット**:
  - 月間50名以上対応の実績あるキャリアアドバイザーが伴走
  - **最大30万円の入社お祝い金**(プッシュしすぎない方針・補助訴求)

### ④ スキルアップサポート
- **内容**: AIリスキリング(経産省認定講座、トスアップ)
- **初期費用**: 0円
- **報酬**: 10万円/件
- **ターゲット**: 雇用保険加入者限定
- **顧客メリット**:
  - **最大48万円の支援金**で実質無料(48万円→0円)
  - 経産省認定カリキュラム
  - 独自の会員サイトでAI基礎〜画像/動画/音楽生成まで学べる
  - 副業に活かせる基礎動画コンテンツ完備

### 運営スタイル
プレゼン担当 → トスアップ(講義・フォローはほぼなし)


## 14. LP改善メモ(2026-04-10確認)

### 良い点
- ファーストビューのイラスト+コピーが共感的
- 4サービスのビジュアル一覧が分かりやすい
- 473.6万円の実績バッジがインパクト大
- お客様の声セクションあり
- ご利用の流れが可視化されている

### 改善すべき点
1. **追従(固定)LINEボタンがない** → スクロール中にCTAが消える
2. **ファーストビューのCTAが弱い** → 473万円を即座に出すべき
3. **Q&Aが全文表示で長い** → アコーディオン式に畳む
4. **最終CTAが「詳細はこちら」で弱い** → 「無料で相談する」「公式LINEで特典受取」に変更
5. **特典訴求(100万円手順書)がLP上にない** → LINE登録動機を強化
